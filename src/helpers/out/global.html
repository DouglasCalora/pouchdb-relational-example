createDatabase(name, aliasopt, options)

This method creates a database and add to the list of databases. Check: https://pouchdb.com/api.html#create_database
Parameters:
Name Type Attributes Description
name string argument of PouchDB can be the name of db too
alias string <optional>
 if you are sending a remote db and wants to give a name to this instance you can send an alias. (alias overrides the param `name`.)
options object PouchDB create Options.
Source:
  • PouchDBSetup.js, line 68
Example
myInstanceOfPouchDBSetup.createDatabase('https://my-remote-db.com', 'myRemoteDB') // the second arg is the alias for this db.
myInstanceOfPouchDBSetup.createDatabase('myDatabase') // if not send an alias name of db will be `myDatabase`
